#### Проект-заготовка под многопользовательскую браузерную игрушку

Есть несколько локаций, на к-ых можно потыкать мышкой, увидеть как меняются данные. 
Для правдоподобности, между пользовательскими событиями работают тайминги.

Стек технологий
- nginx
- mysql, redis (KV-store only)
- php 7, laravel
- bootstrap 3, jquery
- node + socket.io (для модуля аукциона)

Проект разбит на модули ('app\Modules\')  

Схема БД находится в файле 'database\sql\init.sql'
